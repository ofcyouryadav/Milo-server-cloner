## ✨ Features

- 🎭 Clone Roles (with permissions, colors, hoisting)
- 📁 Clone Channels (text, voice, categories, stage, forums + permission overwrites)
- 😀 Clone Emojis
- 🖼️ Clone Server Icon
- 🔀 Full clone or pick individual parts
- 🎨 Colored CLI with interactive menu

---

## 🚀 Setup

### 1. Install dependencies
```bash
npm install
```

### 2. Configure `.env`
```env
TOKEN=your_bot_token_here
```

### 3. Bot permissions needed
Make sure your bot has these permissions in **both** servers:
- Administrator (recommended) OR
- Manage Roles, Manage Channels, Manage Emojis, Ban Members

### 4. Run
```bash
npm start
```

## 📋 Menu Options

| Option | Description |
|--------|-------------|
| 1 | Clone Server (Roles + Channels + Emojis) |
| 2 | Clone Channels Only |
| 3 | Clone Roles Only |
| 4 | Clone Emojis Only |
| 5 | Full Clone (everything including bans) |
| 0 | Exit |
